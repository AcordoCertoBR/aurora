# Icones

para utilizar os icones basta importa-los:

```jsx
import { IconTruck } from '@consumidor-positivo/aurora'
```

Cada icone aceita as seguintes props:

```jsx
type IconProps = {
    size: 'large' | 'default';
    color: 'dark' | 'info' | 'default';
};
```

## Coleção padrão

<IconGallery>
  {Object.entries(iconsDefault).map(([key, Comp]) => (
    <IconItem name={key}>
      <Comp />
    </IconItem>
  ))}
</IconGallery>
