# Dependências

Esse projeto existe algumas dependencias que usamos como base para construção de alguns components e configurações, assim facilitando a construção e aproveitando o que tem já pronto e não ter retrabalho.

## Fonts

No projeto, são utilizadas as fontes [Lexend Deca](https://fontsource.org/fonts/lexend-deca) para títulos e [Source Sans 3](https://fontsource.org/fonts/source-sans-3) para os parágrafos. Para incorporar essas fontes, estamos utilizando a biblioteca Fontsource.

Então é necessário instalar essas fonts no projeto

<p>`npm install @fontsource-variable/lexend-deca`</p>
<p>`npm install @fontsource-variable/source-sans-3`</p>
