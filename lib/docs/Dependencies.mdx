# Dependências

Esse projeto existe algumas dependencias que usamos como base para construção de alguns components e configurações, assim facilitando a construção e aproveitando o que tem já pronto e não ter retrabalho.

## Fonts

No projeto, são utilizadas as fontes do Google [Lexend Deca](https://fonts.google.com/specimen/Lexend+Deca?query=lexend+deca) para títulos e [Source Sans 3](https://fonts.google.com/specimen/Source+Sans+3?query=source+sans+3) para os parágrafos. Para incorporar essas fontes, é necessário instalar essas fonts no projeto:

### import

<p>
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

<link href="https://fonts.googleapis.com/css2?family=Lexend+Deca:wght@100..900&family=Source+Sans+3:ital,wght@0,200..900;1,200..900&display=swap" rel="stylesheet" />
```
</p>

### @import

<p>
```scss
@import url('https://fonts.googleapis.com/css2?family=Lexend+Deca:wght@100..900&family=Source+Sans+3:ital,wght@0,200..900;1,200..900&display=swap');

```
</p>
